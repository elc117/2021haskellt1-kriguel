#### O que faz:
O Programa gera uma bandeira do Brasil simplificada onde a metade da esquerda contém as cores originais e a
metade da direita tem as cores invertidas.

No Shell:  
`$ ghci t1haskell.hs`  
`*Main> main`  

Será gerado o arquivo 'image.svg' com a imagem
